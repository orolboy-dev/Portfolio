CSS Tarixi

Veb-sahifalarga go'zallik bergan tilning yaratilishi

CSS (Cascading Style Sheets) tili ilk bor 1994-yilda CERN tashkilotida Tim Berners-Lee bilan birga ishlagan norvegiyalik mutaxassis Håkon Wium Lie tomonidan taklif etilgan. Dastlab veb-saytlar faqat quruq matn va linklardan iborat bo'lgan, CSS esa ularning dizayni, ranglari va joylashuvini alohida boshqarish uchun yaratilgan.

1996-yilda W3C (World Wide Web Consortium) tomonidan CSS1 standarti rasman e'lon qilindi. CSS-ning joriy etilishi veb-dasturlashda katta inqilob bo'ldi, chunki u sahifaning mazmuni (HTML) va uning ko'rinishini (CSS) bir-biridan butunlay ajratish imkonini berdi. Bu esa kodlarni boshqarishni bir necha barobar osonlashtirdi.

Bugungi zamonaviy internet CSS3 versiyasiga tayanadi. U modul tizimida rivojlanib, veb-dizaynerlarga murakkab animatsiyalar, soyalar, gradientlar hamda Flexbox va Grid kabi moslashuvchan (responsive) joylashuv tizimlarini brauzerning o'zida yaratish imkonini berdi.

Hozirgi kunda CSS shunchaki rang berish vositasi emas, balki veb-ilovalarning foydalanuvchiga taqdim etiladigan vizual asosi hisoblanadi. CSS-siz har qanday zamonaviy sayt o'z jozibasini yo'qotib, oddiy matnli hujjatga aylanib qoladi.
