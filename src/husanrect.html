React Tarixi

Komponentlarga asoslangan interfeyslar inqilobi

React (yoki React.js) kutubxonasi ilk bor 2011-yilda Facebook (hozirgi Meta) kompaniyasining dasturchisi Jordan Walke tomonidan yaratilgan. U birinchi marta Facebook-ning yangiliklar tasmasida (News Feed), keyinchalik esa 2012-yilda Instagram platformasida muvaffaqiyatli qo'llanilgan.

2013-yilning may oyida AQShda bo'lib o'tgan JSConf konferensiyasida React loyihasi Open Source (ochiq kodli) sifatida barcha uchun e'lon qilindi. React veb-dasturlashga mutloqo yangi yo'nalish — Virtual DOM va komponentli (component-based) arxitekturani olib kirdi. Bu texnologiya sahifani to'liq yangilamasdan, faqat o'zgargan qismini juda tezkorlik bilan qayta chizish imkonini berdi.

React doimiy tarzda rivojlanib, veb-ilova yaratish standartlarini o'zgartirdi. Ayniqsa, 2019-yilda taqdim etilgan React Hooks funksiyasi klassli komponentlardan voz kechib, kodni ancha sodda va toza yozish imkonini taqdim etdi. Bugungi kunda esa Next.js kabi frameworklar bilan birgalikda server tomonlama yuklash (SSR) tizimlarida yetakchilik qilmoqda.

Hozirgi kunda React dunyodagi eng ommabop JavaScript kutubxonasi hisoblanadi. Undan nafaqat Meta mahsulotlari, balki Netflix, Airbnb, Uber kabi yirik brendlar ham o'z interfeyslarini qurishda foydalanishadi. Shuningdek, React Native yordamida bitta kod bazasi bilan ham iOS, ham Android mobil ilovalarini yaratish mumkin.
