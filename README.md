# Personal Portfolio Website 🚀

Bu mening shaxsiy portfolio veb-saytim. Sayt orqali men amalga oshirgan loyihalarim, texnologik ko'nikmalarim (skills) va men haqimdagi ma'lumotlar bilan tanishishingiz hamda men bilan bog'lanishingiz mumkin.

## ✨ Xususiyatlari
* **Zamonaviy va Minimalistik Dizayn:** To'q rangli mavzu (Dark Mode) va ko'zga yoqimli vizual effektlar.
* **To'liq Responsive:** Smartfonlar, planshetlar va kompyuter ekranlariga mukammal moslashuvchan interfeys.
* **Tezkor yuklanish:** Statik HTML va Tailwind CSS CDN yordamida yuqori unumdorlik.
* **Aloqa Formasi:** Foydalanuvchilar to'g'ridan-to'g'ri xabar qoldirishlari uchun qulay interfeys.

## 🛠 Ishlatilgan Texnologiyalar
* **HTML5** - Saytning asosiy strukturasi uchun.
* **Tailwind CSS (v4)** - Zamonaviy va moslashuvchan dizayn uslublari uchun.
* **Font Awesome (v6.6.0)** - Sifatli va chiroyli piktogrammalar (icons) uchun.
